Simple Conditional (would)

Afirmativa

Sujeto + verb(Would)+ verb infinitive + complemento

Negativa

Sujeto + verb (Would not)+ verb infinitive + complemento

Interrogativa

Wh + verb (Would) + Sujeto + Infinitive + complemento.

Estructura

Affirmative Negative Question
I I Would run. I Would not run. Would I run?
You You Would run. You Would not run. Would You run?
He/She/It Would run. Would not run. Would He/She/It run?
We/They We Would run. We Would not run. Would We/They run?
